OCNJ Smart Planner V2

Open index.html in a browser.

Included:
- Very visible Family / Solo-Couples planner modes
- Weather mode selector: Sunny, Cloudy, Rainy
- Editable time slots; changing a time auto-shifts later slots
- Activity selectors per slot; changing an activity updates duration/links and shifts following slots
- Weather-aware activity population
- Gas calculator
- Mobile-friendly sticky bottom actions
- Biking link fixed to https://oceancityvacation.com/what-to-do/biking/
- Manco direct website removed; Manco is only included as a Google Maps search option
- Alternate pizza options included: Piccini, Mario's, Three Brothers

Note: Weather mode is manual. It does not fetch live weather, so it works offline and from a local file.
