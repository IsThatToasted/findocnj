OCNJ Family Planner
===================

Open index.html in any browser.

Project files:
- index.html
- assets/styles.css
- assets/app.js

Features:
- Interactive 5-day itinerary tabs
- Clearly defined time blocks
- Activity detail cards with links and Google Maps shortcuts
- Gas calculator
- Rain/heat backup plans
- Saveable packing checklist via localStorage
- Print / Save PDF button

Notes:
- Confirm operating hours, event times, and ticket policies on official sites before traveling.
- Gas calculator defaults are placeholders; enter your actual miles, MPG, and fuel price.
